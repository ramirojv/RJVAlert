Alertas modernas para sistemas administrativos

Modo Oscuro

Diálogos emergentes con estética de tarjeta premium, borde dinámico, brillos y colores según el tipo de mensaje.

  • Colores dinámicos según el icono / tipo (success, error, etc.).
  • Brillo externo que rodea el card siguiendo el movimiento del mouse.
  • Highlight interno suave vinculado al mouse.
  • API simple rjAlert.success/error/warning/info/question/prompt.
  • Compatibilidad directa con Swal.fire si querés drop-in.

API rápida
Atajos por tipo de mensaje 
rjAlert.success("OK", "Guardado correctamente.");
rjAlert.error("Error", "No se pudo completar la operación.");
rjAlert.warning("Atención", "Revisá los datos ingresados.");
rjAlert.info("Información", "Este cambio solo afecta a este módulo.");
rjAlert.question("¿Continuar?", "Esta acción no se puede deshacer.");
Confirmaciones y prompts
/* Confirmación */
rjAlert.question({
  title: "Eliminar registro?",
  text:  "Esta acción no se puede deshacer.",
  showCancelButton: true,
  confirmButtonText: "Sí, eliminar",
  cancelButtonText: "Cancelar"
}).then(res => {
  if (res.isConfirmed) {
    // ...
  }
});

/* Prompt */
rjAlert.prompt({
  title: "Alias de entorno",
  text:  "Ingrese un alias identificador:",
  inputPlaceholder: "Ej: Producción BNA 2025"
}).then(res => {
  if (res.isConfirmed) {
    console.log("Alias:", res.value);
  }
});
